# Live Sports

A Progressive Web App (PWA) for comparing sports betting odds between traditional sportsbooks and prediction markets like Polymarket.

## Getting Started

### Installation

Install dependencies using pnpm:

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Testing

Run the test suite:

```bash
pnpm test
```

### Build

Build the application for production:

```bash
pnpm build
```

## Features

- Compare odds between traditional sportsbooks and prediction markets
- Identify arbitrage opportunities through divergence analysis
- Progressive Web App (PWA) with offline support
- Real-time odds updates from The Odds API and Polymarket
